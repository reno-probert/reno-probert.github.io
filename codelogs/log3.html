CSS Media Queries

What are CSS Media Queries?

CSS Media Queries are a method used in web development and responsive design that allows a website to adapt to various screen sizes, browsers, and other viewing options that a user may utilize to view the site. CSS Media Queries can be coded in to make sure a website can be viewed properly to matter what device the user is using to view it and allows a web designer to create a better experience for users in terms of navigation and accessibility.

What does a CSS Media Querie Look like?

CSS Media Queries are coded into the CSS file of a website and can be used to adapt almost any CSS design element you could want from font-size to background color the site based on how the site is being viewed or interacted with or what it is being viewed on.
