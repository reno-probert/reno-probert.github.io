Possible Color Values

There are multiple ways to define a color that are recognized by CSS and will add the color element to the website. These include:

  • Color Names:

    CSS recognizes 140 different color names that can simply be written into the code to change the color. Simply typing in the name of a color or even a simple variation of the color like "blue", "lightblue", "bluegreen", or "navy" will be recognized by CSS.

  • Hexadecimal (Hex) Values: Hex values are a six-digit code that represents a color. They start with a "#" followed by three pairs of digits representing the red, green, and blue (RGB) components of the color. For example, "#FF0000" represents red, "#00FF00" represents green, and "#0000FF" represents blue.
  • RGB Values: RGB values use the RGB color model to define colors. They are written as "rgb(red, green, blue)" where red, green, and blue are values between 0 and 255. For example, "rgb(255, 0, 0)" represents red, "rgb(0, 255, 0)" represents green, and "rgb(0, 0, 255)" represents blue.
  • RGBA Values: RGBA values are similar to RGB values but include an alpha channel for transparency. They are written as "rgba(red, green, blue, alpha)" where alpha is a value between 0 (fully transparent) and 1 (fully opaque). For example, "rgba(255, 0, 0, 0.5)" represents semi-transparent red.
  • HSL Values: HSL stands for Hue, Saturation, and Lightness. HSL values are written as "hsl(hue, saturation%, lightness%)". Hue is a degree on the color wheel (from 0 to 360), saturation is a percentage (from 0% to 100%), and lightness is also a percentage (from 0% to 100%). For example, "hsl(0, 100%, 50%)" represents red.
