Possible Color Values

There are multiple ways to define a color that are recognized by CSS and will add the color element to the website. These include:

Color Names:

CSS recognizes 140 different color names that can simply be written into the code to change the color. Simply typing in the name of a color or even a simple variation of the color will be recognized by CSS. Examples include:

  • blue
  • lightblue
  • bluegreen
  • navy

Hex Values:

Hex values are a six-digit code that represents a color. They start with a "#" and can be found on various hex code generator websites or image creation softwares. The codes allow a web designer to pick more specific and varied colors. Examples include: "#00FF00" represents green, and "#0000FF" represents blue.

  • #00ff00
  • #ff506f
  • #e1d2b4
  • #c9c6d8

RGB Values:

RGB values use the RGB color model to define colors. They are written as "rgb(red, green, blue)" where red, green, and blue are values between 0 and 255. For example, "rgb(255, 0, 0)" represents red, "rgb(0, 255, 0)" represents green, and "rgb(0, 0, 255)" represents blue.

HSL Values:

HSL stands for Hue, Saturation, and Lightness. HSL values are written as "hsl(hue, saturation%, lightness%)". Hue is a degree on the color wheel (from 0 to 360), saturation is a percentage (from 0% to 100%), and lightness is also a percentage (from 0% to 100%). For example, "hsl(0, 100%, 50%)" represents red.
