<img> vs. background-img

The <img> tag which is used to add images to the HTML of a website. The background-image property is used to add images as a background to an element on a website in CSS. The <img> tag is used to add images that are part of the content of the website, they can have their size altered, descriptions added, and are part of the oveerall flow of the elements on the site. You can add images from files added to your code as well as links to online images.

The background-image property is used to add images that are part of the design of the website, they can't have descriptions added, and rather than going directly as an element into the html, they are a design element added in CSS. These can also have additional CSS modifiers added to them such as:

  • background-size: determines scaling (ex. cover, contain)
  • background-repeat: deterimnes if the image will repeat, forming a pattern
  • background-position: determines where the image will be placed (ex. center, top-right)

Difference Between Resolution and Dimensions in a Bitmap Image

The central difference between dimensions and resolution is that resolution refers to the amount of pixels per inch in an image as a whole, while dimensions refer to the total number of pixels in the width and height of an image.

Image File Types

  • JPEG: Good for both photographs and images especially large sizes. Compression can reducee file size but may result in some loss of quality, especially if uploaded (and therefore compressed) multiple times.
  • PNG: Good for images with transparent backgrounds and/or sharp, clean edges. Good for logos and icons and uses lossless compression, preserving image quality.
  • GIF: Good for simple animations. Uses a limited amount of colors that can make the image look less detailed or of lesser quality.
  • SVG: A vector format that is good for logos, icons, and illustrations that need to scale without losing quality. Not as good for things like photographs which would not be in a vector format.
  • WebP: Google made image format that has better compression rates than JPEG and PNG while maintaining quality. Very good for web use.
