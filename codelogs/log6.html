Reference Quest

How Different Coding Sites Explain the @media CSS Rule

For this project I searched for the CSS @media rule using the provided search bar on the following four coding reference sites:

  • Mozilla Developer Network aka MDN
  • W3Schools
  • CSS-Tricks
  • Free Code Camp

What's the Difference?

Term Searchability

All of the sites provided a search bar for looking up specific terms, however some of the results were easier to parse through than others. W3Schools and MDN took me straight to an easy to use page that included both detailed descriptions as well as examples of what the code should look like. CSS-Tricks also provided an in-depth explanation of the @media rule, but unfortunately the @media did not show up in their index of @ CSS terms under "M" so I did have to search around more. Free Code Camp provided a variety of articles and tutorials on the @media rule, but the results page was less focused and harder to find specific information as it is made up of different articles by different authors rather than one standard page of definitions of @media rules.

Clarity

All four sites provided clear explanations of the @media rule, but some were more beginner friendly than others. W3Schools provided an easy to understand explanation of the @media rule, along with simple examples that made it easy to grasp the concept quickly. MDN also provided a clear explanation, but it was more detailed and technical, which may be better suited for those with some prior knowledge of CSS. CSS-Tricks offered similarly detailed explanations to the prior two and included code samples for a large number of @media adjustments which would allow for a user to simply copy and paste or see very clearly what was required of them. Free Code Camp provided a variety of articles that catered to different skill levels, but the quality and clarity varied depending on the author.

Unique Features

Each site had its own unique features that set it apart from the others. W3Schools had interactive examples that allowed users to edit and test the code directly on the page. MDNincludes many links and ways to get more in-depth information in addition to a large section on browser compatibility which was either not present in the other sites or far less detailed. CSS-Tricks has many different articles and tutorials as well as the previously mentioned clear code samples for the @media rule. Free Code Camp had a multitiude of community created articles, with user-contributed articles and tutorials showing more of the real-world applications of the @media rule.

What Does a Good Reference Site Need?

Based on my experience in using a variety of reference sites to clarify things in this class, I feel the following things every reference site should have:

  • A clear and easy to use search function that takes you directly to the page you need based on the term you are looking for.
  • Clear explanations that are easy to understand for beginners but with easily accessible detailed information for advanced users.
  • Examples that are easy to follow and provide direct samples of the actual code that will be needed.
  • Web design that allows for maximum readability. Don't hide great information in a wall of text.

What Stands Out?

Overall, all of the sites did well at providing the needed information. W3 Schools has a very strong design that provides clear information in a readable format that makes it very simple to navigate. CSS Tricks provides very clear samples of specific code for the different CSS properties that can be set up with the @media rule. MDN has an incredibly detailed database with a multitude of links, though it does suffer a little from less readability and design hierarchy than you would see in some of the other sites. To me the weakest references came from Free Code Camp as the community driven approach provides a lot of articles, you have to sift through them to find the information you actually need.

If I were to rank these sites based on how helpful they are to myself, a beginning coder, I would order them as:

  1. W3Schools
  2. CSS-Tricks
  3. MDN
  4. Free Code Camp
